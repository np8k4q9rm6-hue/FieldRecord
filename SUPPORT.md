# FieldRecord – Support

Welcome to the FieldRecord support page by CodePeregrine LLC.

## Overview
FieldRecord is designed for on‑device item management with privacy and chain‑of‑custody in mind. We do **not collect or store** your data to ensure your privacy is always protected.

## Getting Help
- **In-app:** Settings → Support  
- **Online:** GitHub repository  
  - Issues: [https://github.com/np8k4q9rm6-hue/itemLogPro/issues](https://github.com/np8k4q9rm6-hue/fieldrecord/issues)  
  - Security Policy: [https://github.com/np8k4q9rm6-hue/itemLogPro/blob/main/SECURITY.md](https://github.com/np8k4q9rm6-hue/fieldrecord/blob/main/SECURITY.md)

## Reporting Issues
- Open a GitHub issue including steps to reproduce, expected vs. actual behavior, and device/iOS version.  
- **Do not include sensitive or personally identifiable information.**  
- If logs are needed, **sanitize them carefully before sharing** to avoid exposing private data.

## Feature Requests
- Submit GitHub issues labeled `enhancement`.  
- Provide a clear description of the use case and the value it adds.

## Emergency Handling
If you believe there is a critical security issue, please refer to our [Security Policy](https://github.com/np8k4q9rm6-hue/fieldrecord/blob/main/SECURITY.md) for responsible disclosure instructions.
